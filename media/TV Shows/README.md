# Beispiel-Ordnerstruktur

Platziere hier deine TV-Serien nach folgendem Schema:

```
TV Shows/
├── Breaking Bad/
│   ├── Season 01/
│   │   ├── breaking.bad.s01e01.720p.mkv
│   │   └── breaking.bad.s01e02.720p.mkv
│   └── Season 02/
│       └── breaking.bad.s02e01.720p.mkv
└── The Office/
    └── Season 01/
        └── the.office.s01e01.mkv
```

Das Tool wird die Episoden automatisch basierend auf TMDB umbenennen.
